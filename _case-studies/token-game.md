## A small experiment with Claude Code

I&nbsp;wanted to&nbsp;experiment with Claude Code and build something small, fast, and fun. A&nbsp;quiz felt like the simplest format.

Since color tokens were already part of&nbsp;my&nbsp;daily work in&nbsp;our design system, they became the perfect subject.

What started as&nbsp;an&nbsp;experiment became a&nbsp;lightweight game. Simple enough to&nbsp;play in&nbsp;a&nbsp;few minutes, but specific enough to&nbsp;make you think twice about token names.

## From AI-generated to&nbsp;intentional

The first version worked, but looked generic&nbsp;&mdash; a&nbsp;bit too AI-generated. I&nbsp;refined the layout and interactions to&nbsp;make it&nbsp;feel more intentional, focused, and game-like.

To&nbsp;make the feedback more memorable, I&nbsp;added reaction GIFs. It&nbsp;made the quiz feel less like a&nbsp;test and more like a&nbsp;tiny game.

## Sharing it&nbsp;with the team

I&nbsp;shared the game with the design team and presented it&nbsp;during one of&nbsp;our meetings. We&nbsp;played it&nbsp;together and compared scores.

What started as&nbsp;a&nbsp;fun side experiment unexpectedly became a&nbsp;playful way to&nbsp;learn and remember our design tokens.
