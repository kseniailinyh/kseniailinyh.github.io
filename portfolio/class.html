Class is a platform for students and teachers which provides communication during the remote educational process. Class was created for public schools in Russia that were forced to switch to the distant format because of COVID-19.

Public schools were not ready for distant format. Eventually, they had zoom and other tools for video calls, but there was another huge problem: how to gather and review the homework from students.

Class solves this problem. Students get homework → do it on paper → took pictures and send them to a teacher → the teacher reviews it and gives a mark.

I built the UI from the scratch. There is a mobile web interface for students, which allows them to get the homework, do it on paper, take a photo and send it to the teacher for review.

Also, there is a desktop web interface for teachers. The first product manager's idea for this UI was to build a messenger where teachers will chat with students:

But, after studying the result of the research, I proposed a completely different idea for the teacher's UI, convinced the product manager, and validated it through user testing.
