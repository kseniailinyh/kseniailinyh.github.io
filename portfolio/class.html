Kontur.Class

Kontur.Class is a platform for students and teachers which provides communication for the remote educational process. It fits for schools that were forced to switch to the distant format because of COVID-19. Kontur.Class allows to avoid interruptions in education and helps teachers organize their work.

There are 2 interfaces: for teachers and students. Students work on a mobile web interface, which allows them to get the homework and send completed tasks to the teacher for review.

Mobile interface for students:

Teachers get homework from students in the web interface, review them, leave comments and notes, and give marks. There is also a chat with students, homework designer, and much more.

I prepared a comprehensive description with all possible states for every part of the interface. It helped me not forget to design some details and helped developers better understand the task.

I was on the project from the very beginning and made the first low-fidelity prototypes to visualize the whole idea. Also prepared interactive prototypes for testing hypotheses. After testing I persuade the team to change the conception:

– to make an inbox with homework for a teacher instead of a messenger with students

– and, as a result, to make homework the main entity. These changes influenced crucially the final interface.

Now the product is in the testing stage in a few public schools in Russia.
